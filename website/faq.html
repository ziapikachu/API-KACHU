Les questions fréquentes

♦ Quel format dois-je suivre pour créer un ticket ?

- Je souhaite créer un ticket relatif à un bug

Le titre doit contenir une balise [BUG], une explication rapide du bug et ajoutez le label correspondant.

Suivez ensuite ce template pour décrire votre bug :

  • Etape pour reproduire le bug
  • Ce que je vois
  • Ce que j'aurai du voir
  • Explication du bug
  • Sur quel OS vous avez eu ce bug

- Je souhaite créer un ticket pour proposer une amélioration

Le titre doit contenir une balise [ENHANCEMENT], ce que vous souhaitez améliorer et ajoutez le label correspondant.

Suivez ensuite ce template pour décrire votre amélioration :

  • Ce que vous souhaiter améliorer
  • Ce que vous allez ajouter (description et explication de l'ajout)
  • Ce qui n'allait pas

- Je souhaite créer un ticket pour recevoir de l'aide

Le titre doit contenir une balise [HELP WANTED], ce que vous souhaitez obtenir comme explication et ajoutez le label correspondant.

Suivez ensuite ce template pour décrire votre demande :

  • Ce que vous n'avez pas compris
  • Ce que vous souhaitez que l'on vous explique (en détail)

- Je souhaite poser une question

Le titre doit contenir une balise [QUESTION], le sujet de votre question et ajoutez le label correspondant.

  • Votre question

Tous les tickets créés qui ne suivront pas ces templates ne seront pas pris en compte.

♦ Si je découvre un bug, et que j'arrive à le corriger, comment je peux proposer une correction ?

Vous pouvez faire un pull request à cette adresse avec votre patch.
